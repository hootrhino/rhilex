# 遥测统计

## 配置
```ini
[plugin.telemetry]
enable=true # 启用
addr="remote.site:8080"
```

## 接口
UDP传输
```json
{
    "arch": "amd64",
    "os": "windows",
    "start_at": "2024-08-11 17:50:29",
    "deviceId": "000000000001",
    "mac": "A8:A1:59:2E:A2:D9",
    "admin": "WWHAI@RHILEX-DEVELOP"
}
```

## 注意
该插件需要配合用户协议使用，目前对于私人信息采集有一个规范协议。防止法律风险。
