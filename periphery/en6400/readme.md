## EN6400
畅维通达 EN 系列网关是一款面向工业、农业、交通运输、科研等领域，集数据采集、
通讯、边缘计算于一体的通用设备。设备的操作系统采用 Ubuntu18.04.6 LTS，用户可
以基于 Node-RED 快捷的二次开发。
